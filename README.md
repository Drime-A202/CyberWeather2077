# CyberWeather2077 · 中国天气地图 & 数据诗学艺术

> 一个兼具实用与美感的交互式天气可视化项目
> 实时天气地图 + 赛博朋克风格粒子艺术 —— 让数据拥有情绪。

---

## 项目简介

本项目包含两个独立又统一的体验：

1. **天气地图**（/）：基于 Flask + ECharts 的中国交互式天气地图，点击城市查看实时天气和 7 天预报。
2. **数据诗学艺术**（/art）：将天气数据（温度、风速、天气代码）实时映射为动态粒子系统，每个城市拥有独特的视觉"情绪"。

> 本项目是"网站程序设计基础"课程的结课作业，也是探索"数据艺术"的一次尝试。

---

## 核心功能

| 功能 | 说明 |
| :--- | :--- |
| 交互式中国地图 | ECharts 渲染，支持缩放/拖拽，城市标记颜色随温度变化 |
| 实时天气数据 | 接入 Open-Meteo API，展示温度、体感、湿度、风速、气压等 |
| 动态艺术粒子 | 根据温度、风速、天气代码动态改变粒子的颜色、速度、形态（晴/雨/雪/雷暴） |
| 7 天预报 | 侧边栏展示未来一周天气趋势 |
| 鼠标交互 | 悬停查看数据浮窗，点击切换城市，粒子受鼠标扰动 |
| 赛博朋克 UI | 霓虹配色、动态扫描线、系统日志，沉浸式体验 |

---

## 技术栈

- 后端：Python 3.9+ / Flask 2.0
- 前端：ECharts 5 / HTML5 / CSS3 / Vanilla JavaScript
- 数据源：Open-Meteo API（无需 API Key）
- 其他：Requests（数据抓取）

## 快速开始（本地运行）

1. 克隆仓库
   ```
   git clone https://github.com/Drime-A202/CyberWeather2077.git
   cd CyberWeather2077
   ```

2. 安装依赖
   ```
   pip install -r requirements.txt
   ```

3. 启动应用
   ```
   python app.py
   ```

4. 访问
   - 天气地图：http://127.0.0.1:5000/
   - 艺术粒子页：http://127.0.0.1:5000/art

---

## 使用指南

- 地图页：鼠标悬停城市 → 浮窗显示实时数据；点击城市 → 右侧面板更新详情。
- 艺术页：下拉菜单切换城市 → 粒子颜色、运动、形态随之变化；移动鼠标扰动粒子。
