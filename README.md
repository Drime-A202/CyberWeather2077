# 🌦️ CYBER_WEATHER · 中国天气地图 (Flask + ECharts)

> 网站程序设计基础 · 结课作业 —— 一个具有赛博朋克风格的交互式中国天气可视化地图。

## 📖 项目简介

本项目基于 **Flask** 后端和 **ECharts** 前端，调用 Open-Meteo 开源天气 API，实现了中国主要城市实时天气数据的可视化展示。用户可以通过点击地图上的城市标记，查看该城市的实时气温、湿度、风力以及未来 7 天的天气预报。

## ✨ 核心功能

- 🗺️ **交互式中国地图**：使用 ECharts 渲染，支持缩放、拖拽。
- 🌡️ **实时天气数据**：集成 Open-Meteo API，展示温度、体感温度、湿度、气压、风速等。
- 🎨 **动态视觉反馈**：城市标记根据温度高低呈现不同颜色（冷蓝 → 暖红），天气状况改变界面主题色。
- 📅 **7 天预报**：侧边栏展示未来一周的天气趋势。
- 📟 **赛博朋克 UI**：仿终端风格，带有动态扫描线和系统日志。

## 🛠️ 技术栈

- **后端**：Python 3.x + Flask
- **前端**：ECharts 5, HTML5, CSS3, JavaScript
- **数据源**：[Open-Meteo API](https://open-meteo.com/)（无需 API Key）
- **其他**：Requests 库用于数据抓取
